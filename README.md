# Salil-Chand
Repository for my git hosted portfolio which includes some projects.

## Repository Structure
* **`index.html`:** The main file containing the website's structure, URLs and content.
* **`README.md`:** This file, providing an overview of the portfolio and repository.
* **`images/`:** Directory storing all visual assets / images.
  * *Contains:* `your-photo.jpg`, `heatmap.png`, and other project visualizations.
* **`projects/`** : Directory for data analysis scripts and notebooks.
  * *Contains:* Project 1: `LondonBikeShare.ipynb`.
